# @krititva/api-client

Generated TypeScript client for the Krititva API. Codegen wires up in **M1.T3** when the artifacts endpoints ship an OpenAPI spec worth pinning.

Until then, `src/index.ts` exports a placeholder so the workspace resolves and `@krititva/web` can `import { createApiClient } from "@krititva/api-client"`. Do not add hand-written methods here — real methods come from the generator.
